<details>
<summary>⚙️ <b>How to activate the Snake (click to expand)</b></summary>

<br/>

**Step 1 — Enable workflow permissions**
> Go to your `Eng-Mounir` repo → **Settings** → **Actions** → **General** → **Workflow permissions** → select **"Read and write permissions"** → Save

**Step 2 — Create the workflow file**

Inside the `Eng-Mounir` repo, create this file:
`.github/workflows/snake.yml`

Paste this content:

```yaml
name: Generate Snake Animation

on:
  schedule:
    - cron: "0 */12 * * *"
  workflow_dispatch:

jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk/svg-only@v3
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark

      - uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

**Step 3 — Run it manually**
> Go to **Actions** tab → **Generate Snake Animation** → **Run workflow**

✅ Done! The snake will auto-update every 12 hours.
